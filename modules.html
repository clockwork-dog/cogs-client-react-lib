    • Defined in providers/CogsConnectionProvider.tsx:66

    Create a persistent connection to COGS which can be accessed with useCogsConnection()

    Parameters

    • __namedParameters: { audioPlayer?: boolean; children: React.ReactNode; hostname?: string; port?: number; videoPlayer?: boolean }
      • Optional audioPlayer?: boolean

        Creates a CogsAudioPlayer than can be accessed with useAudioPlayer()

        Note: If unset, the audio player will remain active.

      • children: React.ReactNode
      • Optional hostname?: string
      • Optional port?: number
      • Optional videoPlayer?: boolean

        Creates a CogsVideoPlayer than can be accessed with useVideoPlayer()

        Note: If unset, the video player will remain active.

        Example:

        function MyComponent() {
  const cogsConnection = useCogsConnection();
  const numberOfPlayers = useCogsConfig(cogsConnection)['Number of players'];

  return <div>Players: {numberOfPlayers}</div>;
}

function App() {
  return <CogsConnectionProvider audioPlayer videoPlayer>
    <MyComponent />
  <CogsConnectionProvider/>;
}
